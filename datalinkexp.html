Data Link Layer

Ethernet -

Ethernet defines how devices on the same network connect and send data to each other using cables or switches.

PPP -

Stands for Point-to-Point Protocol. It allows two computers to directly connect and send data over a serial link, like a phone line, cable, or internet connection.

IPsec -

A network switch is a piece of hardware component that sends and receives data within a local area network (LAN) to enable them to communicate and share resources efficiently.

Bridge -

A bridge is a device in the data link layer that connects two or more network segments and forwards data only to the part of the network where it's needed.
