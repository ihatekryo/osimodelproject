Applications Layer

HTTP -

Stands for HyperText Transfer Protocol. It is the set of rules that allows web browsers and servers to communicate. It's how your browser requests a webpage and how the server sends it back. It's not used too much any more as it's been replaced by HTTPS (HyperText Transfer Protocol Secure).

FTP -

Stands for File Transfer Protocol. It is a standard way for computers to send and receive files over the internet. It lets you upload files from your computer to a server or download files from a server to your computer and it was replaced by SFTP because of the lack of encryption and security.

---------------------------------------------------------

IRC -

Stands for Internet Relay Chat. It's a system that lets people chat with each other in real time over the internet, usually in group chat rooms or private messages. Still used today but only by some people as it's not as secure as other platforms like Discord.

SSH -

Stands for Secure Shell. It is a network protocol that lets you safely connect to another computer and transfer files over a network. It's mainly used to log in to remote servers and run commands securely, since all the data is encrypted. SSH is also still used today!

DNS -

Stands for Domain Name System. DNS is like the phonebook of the internet. It is able to convert names like "Google.com" into IP adressess to be used by computers to find other computers.

URL -

Stands for Uniform Resource Locator. A URL is the address of something on the internet. It's what tells your browser where to find a resource (webpage, image, or file).
