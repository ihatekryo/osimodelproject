Network Layer

IP -

Stands for Internet Protocol. It is the system that gives every device on a network an address and makes sure data knows where to go and how to get there.

ICMP -

Stands for Internet Control Message Protocol. It is a network protocol used by computers to send error messages and status updates about data travel.

IPsec -

Stands for Internet Protocol Security. It is a set of tools that makes network communication private and secure by encrypting and checking the data sent between computers over the internet.

IGMP -

Stands for Internet Group Management Protocol. It's a network protocol that lets computers tell a router they want to join or leave a multicast group. A multicast group is a group of computers on a network that want to get the same data, like a video stream or stock updates.
