Transport Layer

TCP -

Stands for Transmision Control Protocol. TCP is a fundamental internet protocol that provides reliable, ordered, and error-checked delivery of data between applications on different hosts. TCP does this by doing something called a three-way handshake in order to set up a secure and stable connection, making sure both computers are ready to communicate.

UDP -

Stands for User Datagram Protocol. It's a connectionless, unreliable transport layer protocol used in computer networks for fast, low-overhead data transmission that is sent straight to a socket. Streaming services like Netflix and Youtube use UDP.
