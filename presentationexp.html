Presentation Layer

(The application layer is the "End User Layer." It provides the interface between user applications and the network, allowing things like email, web browsing, and file transfering. The application layer interacts with the lower layers to move data generated by the end user, like an email, across the network.)

SSL -

Stands for Secure Socket Layer. SSL is a security technology that encrypts data sent between a web browser and a website. It makes sure information like passwords, credit cards, or messages can't be easily read by others while traveling across the internet.

SSH -

Stands for Secure Shell. It is a network protocol that lets you safely connect to another computer and transfer files over a network. It's mainly used to log in to remote servers and run commands securely, since all the data is encrypted. SSH is also still used today!

IMAP -

Stands for Internet Message Access Protocol. It's a way to read email directly from the mail server. It lets you see your emails on multiple devices (like phone, laptop, tablet) without downloading them permanently, since the messages stay stored on the server.

FTP -

Stands for File Transfer Protocol. It is a standard way for computers to send and receive files over the internet. It lets you upload files from your computer to a server or download files from a server to your computer and it was replaced by SFTP because of the lack of encryption and security.

MPEG -

Stands for Moving Picture Experts Group. MPEG is a set of standards for compressing audio and video files so they take up less space and can be shared or streamed more easily. It's also a file format!

JPEG -

Stands for Joint Photographic Experts Group. JPEG is a common format for compressing and saving pictures so they use less storage while still keeping good quality. It's the format most photos use on cameras, phones, and the web. A cool fact is that a JPEG image sometimes loses quality when you edit and save it many times!
